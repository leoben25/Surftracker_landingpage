# SurfTracker Landing - Figma Style

Proyecto React + Vite reconstruido con la guía visual de Figma:

- Tipografía: Sora para títulos, Inter para texto.
- Paleta: Navy, blue, sky, neutrals.
- Radios, sombras y UI elements similares al style guide.
- Animaciones suaves y hover states.

## Ejecutar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy en Vercel

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`
